# Tic Tac Toe

Implementation of popular Tic Tac Toe game done with Flutter engine

## Features

- Fully working game
- Support of any board size
- Offline mode on one device (take turns!)

## Planned Features

- Offline mode vs Computer
- - Easy mode
- - Hard mode
- Online mode

- Profile
- Game History
- Statistics
- Custom Symbols
- Cancel turns
